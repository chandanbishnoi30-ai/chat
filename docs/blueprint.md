# **App Name**: ConnectNow

## Core Features:

- Real-Time Messaging: Send and receive text, voice, and video messages instantly.
- Multimedia Sharing: Share photos, videos, documents, and locations with contacts.
- Group Chats: Create and manage groups for communicating with multiple people simultaneously.
- End-to-End Encryption: Secure all communications with end-to-end encryption for enhanced privacy.
- Contact Management: Manage contacts, add new contacts, and block unwanted contacts.
- Status Updates: Share temporary status updates with contacts (text, photo, or video).
- AI-Powered Smart Reply Suggestions: Suggest relevant replies to incoming messages, acting as a tool to assist the user. The LLM decides when suggestions are helpful.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and reliability.
- Background color: A light blue-gray (#E5F1F5) to provide a clean and modern feel.
- Accent color: A contrasting orange (#F2994A) to highlight important actions and notifications.
- Body and headline font: 'PT Sans' for a modern and friendly look.
- Use clean, minimalist icons for easy navigation.
- Intuitive layout with a clear separation between chats, contacts, and settings.
- Subtle animations for message delivery and UI transitions to enhance user experience.
